# [SchmitzIT] Custom Tabs

Adds configurable custom tabs to XenForo Resource Manager resources, with per-tab permission control and a rich-text editor for walkthrough content.

## Requirements

- XenForo 2.4.0+
- XenForo Resource Manager 2.4.0+

## Installation

1. Copy the contents of the `upload/` folder into the root of your XenForo installation.
2. In the Admin Control Panel, go to **Add-ons** and click **Install/Upgrade from Archive**, or use the CLI installer.
3. Follow the on-screen prompts to complete the installation.

## Features

- Add custom tabs to any Resource Manager resource
- Per-tab permission control to restrict visibility
- Rich-text editor for tab content (walkthrough-style)
