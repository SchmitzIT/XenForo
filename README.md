# SchmitzIT XenForo Add-ons

This repository contains XenForo add-ons developed by SchmitzIT.

## Add-ons

| Folder | Add-on | Version | Description |
|--------|--------|---------|-------------|
| [SchmitzIT - CustomTabs](./SchmitzIT%20-%20CustomTabs) | [SchmitzIT] Custom Tabs | 1.0.0 | Adds configurable custom tabs to XenForo Resource Manager resources, with per-tab permission control and a rich-text editor for walkthrough content. |

## Requirements

Each add-on lists its own requirements in its `addon.json`. In general:

- XenForo 2.4.0+

## Installation

Copy the contents of the add-on's `upload/` folder into the root of your XenForo installation, then install the add-on via the XenForo Admin Control Panel.
